Installer dans Outlook (bureau)

  1. Ouvrir ce fichier dans le navigateur, sélectionner l'aperçu ci-dessus (juste le bloc), Ctrl+C.
  2. Dans Outlook : Fichier → Options → Courrier → Signatures…
  3. Cliquer Nouveau, nommer « OZ », puis coller dans la zone d'édition avec Ctrl+V.
  4. Choisir cette signature par défaut pour les Nouveaux messages et les Réponses, puis OK.

Outlook web / nouveau Outlook : Paramètres → Courrier → Rédiger et répondre → coller le bloc dans la signature.

Bon à savoir sur le logo

Le logo est directement intégré au bloc : quand tu colles la signature dans Outlook, l'image devient une pièce embarquée dans le message — elle s'affiche chez tous les destinataires, sans dépendre d'internet ni du site. Le fichier logo-oz-signature.png est aussi fourni si tu préfères l'insérer à la main.
