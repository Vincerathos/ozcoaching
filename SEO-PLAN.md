## 2. À FAIRE avant la mise en ligne (technique)

- [ ] **Acheter le nom de domaine** (idéalement `ozcoaching.fr`). Remplacer `https://www.ozcoaching.fr/` partout dans les fichiers si le domaine diffère.
- [ ] **Hébergement avec HTTPS** (certificat SSL obligatoire pour le SEO). Options simples et gratuites/peu chères : Netlify, Vercel, Cloudflare Pages, ou o2switch/OVH.
- [ ] **Ajouter les vrais visuels** (et compresser en WebP < 200 Ko) :
  - `aurelia-grino.jpg` — portrait
  - `logo-oz-coaching.png` — logo officiel
  - `og-oz-coaching.jpg` — image de partage (1200×630 px)
- [ ] **Connecter l'agenda de RDV** (Calendly / Cal.com) sur le bouton "Choisir un créneau".
- [ ] Compléter l'**hébergeur** dans `mentions-legales.html`.
- [ ] Vérifier que le **mail `contact@ozcoaching.fr`** existe (ou remplacer par le vrai).

## 3. Le levier n°1 pour un coach local : Google Business Profile 🔝

> **C'est de loin l'action la plus rentable.** C'est ce qui fait apparaître Aurélia dans le "pack local" (la carte Google) et sur Google Maps.

- [ ] Créer une **fiche Google Business Profile** gratuite : https://business.google.com
- [ ] Catégorie principale : **"Coach"** / "Consultant en recrutement" / "Service de coaching".
- [ ] Zone de service : Montpellier, Nîmes, Occitanie (possibilité de masquer l'adresse si elle reçoit à distance).
- [ ] Ajouter : photos (portrait, logo), description avec mots-clés, horaires, lien vers le site, lien RDV.
- [ ] **Collecter des avis Google** : demander à chaque coaché satisfait un avis (les 12 recommandations LinkedIn existantes = un vivier idéal à réactiver). Viser 10+ avis 5★ → impact énorme sur le classement local.
- [ ] Publier des "posts" Google réguliers (lives, conseils) → signal de fraîcheur.

**Bonus annuaires (citations NAP — Nom/Adresse/Téléphone cohérents partout) :**
PagesJaunes, Justacoté, malt.fr, et annuaires de coachs (ICF France, etc.).

## 4. Soumettre le site à Google

- [ ] Créer un compte **Google Search Console** (https://search.google.com/search-console) → valider le domaine.
- [ ] Y soumettre le `sitemap.xml`.
- [ ] Demander l'indexation de la page d'accueil.
- [ ] (Optionnel) **Bing Webmaster Tools** — même logique, capte aussi ChatGPT/Copilot.

Idées d'articles (réutiliser ses posts existants) :
1. "Les 'trous' dans un CV : comment les raconter en entretien" (son post pause pro)
